# Club Stats

## Q1: How would that 1 odd guy perform the task?

1. Visit the rankings page: https://www.ea.com/games/nhl/nhl-21/pro-clubs/rankings
2. Click the Xbox button
3. Search the club name: 239 GLOBETROTTERS
4. Click the club name (now showing club overview page)
5. Click view all match history link
6. Click the GameMode button and set it to "Private Games"
** Note: This page only has a 5 game history
7. Record the team stats and player stats for each game

## Q2: How is the data for these matches used and download as?

1. season 19 stats.xlsx is an example spreadsheet for storing the statistics. 
2. From our conversation, I remember that all we really need to do is download the stats and then the raw data can be formatted into usable form.
3. The spread sheet contains the following information:  
    1. Season names tab shows the team owner's name and the name for their club
    2. The stats enter page tab shows the stats per game for each player (see ea-skater-stats-page.png for visual)
